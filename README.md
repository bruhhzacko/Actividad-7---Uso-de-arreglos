# Actividad-7---Uso-de-arreglos

Este programa en C++ nos permite ingresar y almacenar las calificaciones de varios estudiantes usando arreglos. Calcula el promedio de cada estudiante, el promedio general del grupo, y determina la nota más alta y la más baja. Además, clasifica a los estudiantes en aprobados y reprobados según su promedio, mostrando todos los resultados en pantalla de forma organizada.


**usos de arreglos** :
Se utilizaron dos arreglos en el programa. El primero almacena las calificaciones de los estudiantes, guardando cuatro notas para cada uno. El segundo arreglo se utiliza para guardar el promedio de cada estudiante.
